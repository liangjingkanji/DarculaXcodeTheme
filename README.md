# Darcula_Xcode
Darcula of Xcode, Xcode的`Darcula`编辑器主题



## 预览

![image-20210216120613382](https://i.imgur.com/2XGnJkG.png)



## 安装

放置到`~/Library/Developer/Xcode/UserData/FontAndColorThemes/`路径下



然后在Xcode中选中主题
